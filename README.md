## Tentang Pegawat

Pegawat adalah aplikasi WhatsApp Gateway modern untuk mengelola banyak sesi WhatsApp, auto reply, broadcast, scheduler, webhook, dan dokumentasi API dalam satu dashboard. Proyek ini diarahkan untuk kebutuhan otomasi, operasional, layanan pelanggan, dan eksperimen integrasi berbasis REST API.

## Fitur Utama

- Multi-session WhatsApp dashboard.
- Auto reply dan bot command management.
- Broadcast dan scheduler pesan.
- Dokumentasi API dan Swagger UI.
- Manajemen kontak, grup, media, dan webhook.
- Self-hosted sehingga data tetap berada di infrastruktur Anda.

## Instalasi Singkat

```bash

cd pegawat
npm install
cp .env.example .env
npm run db:push
npm run dev
```
